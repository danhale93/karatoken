## 🚀 Quick Start

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation
```bash
# Clone or navigate to the project
cd /workspace

# Install dependencies (if needed)
npm install -g ts-node typescript @types/node

# Make sure all files are in place
ls -la demo/
```
